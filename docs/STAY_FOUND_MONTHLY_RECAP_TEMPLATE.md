# Stay Found Monthly Recap Template

Status: active template
Owner: Client Success
Reviewer: Auditor
Purpose: send a short monthly note that proves GMF kept the client's Google-facing footprint fresh without forcing the owner into a dashboard.

## When To Use

Use this for every active Stay Found client once per month.

The recap should be short, factual, and calm. Do not invent ranking claims, search volume, AI visibility scores, or review numbers that are not supported by source data.

## Inputs

- client name and profile link
- current Google Business Profile access state
- profile changes or drift noticed this month
- review path status
- new review count or unanswered-review status, when available
- website/contact info status
- any client approvals needed

## Client Recap Format

Subject:

```text
Your monthly Stay Found update
```

Body:

```text
Hi {{owner_first_name}},

Here is your quick Stay Found update for {{business_name}}.

What we checked:
- Google Business Profile basics
- Review link and review activity
- Website/contact info consistency
- Public trust signals that can get stale over time

What changed:
- {{one_to_three_factual_changes_or_no_major_changes}}

What needs attention:
- {{one_to_three_attention_items_or_no_action_needed}}

What GMF is doing next:
- {{next_gmf_action}}

What we need from you:
- {{client_request_or_none}}

Short version: {{one_sentence_plain_language_summary}}

Thanks,
GetMeFound
```

## Agent Handoff

Profile Manager supplies the factual visibility notes.

Reviews Manager supplies review and unanswered-review status.

Client Success writes the recap in plain business-owner language.

Auditor checks that all claims are supported and that no public profile edit is implied without approval.

Manager asks Mike only if the recap needs a sensitive client request, a paid upsell recommendation, or approval for a public-facing change.

## Auditor Checklist

- The recap is less than 250 words unless there is a real issue.
- Every number has a source.
- The note avoids promises like "rank higher" or "guaranteed visibility."
- Any client request is clear and small.
- Any risky public profile change is routed for approval before action.
